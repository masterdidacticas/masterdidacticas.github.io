¿Qué es <HTML>?

HTML (HyperText Markup Language) es el lenguaje que se usa para estructurar páginas web.

Explora el video y aprende sobre HTML.

¿Para qué sirve?


Gracias a HTML puedes crear diferentes bloques que componen una página web como encabezados, párrafos, imágenes, enlaces y más. Se utiliza junto con CSS para el diseño visual y con JavaScript para funcionalidades interactivas.

Esturctura y elementos de HTML

Estructura HTML básica

<!DOCTYPE html>
              <html>
                <head>
                  <meta charset="UTF-8">
                  <title>Mi Página</title>
                </head>
                <body>
                  <h1>Hola mundo</h1>
                </body>
              </html>

Elementos esenciales

  1. Encabezados: <h1> a <h6>
  2. Párrafos: <p>
  3. Imágenes: <img>
  4. Listas: <ul>, <ol>, <li>
  5. Enlaces: <a>
  6. Tablas: <table>, <tr>, <td>
  7. Formularios: <form>, <input>

Tabla de Etiquetas HTML

Etiqueta Descripción Ejemplo
