¿Qué es <HTML>?

HTML (HyperText Markup Language, en español, Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear la estructura de las páginas web.   Con HTML puedes definir encabezados, párrafos, listas, enlaces, imágenes, formularios y muchos otros elementos que componen el contenido de un sitio web.   Es la base sobre la que se construyen todas las páginas en Internet

¿Para qué sirve HTML?

Gracias a HTML puedes:

  • Crear diferentes bloques de contenido

    • Como encabezados, párrafos, listas, tablas, imágenes, enlaces y más, que forman la estructura básica de una página web.


  • Organizar el contenido de forma jerárquica

    • Usando etiquetas como <h1> a <h6> para títulos, y <section>, <article> o <div> para dividir el contenido.


  • Insertar elementos multimedia

    • Como imágenes (<img>), videos (<video>) y audio (<audio>) para enriquecer la experiencia del usuario.


  • Crear formularios interactivos

    • Con campos de texto, botones, casillas de verificación y más, usando la etiqueta <form> y sus elementos asociados.


  • Enlazar páginas y recursos

    • Mediante la etiqueta <a>, puedes conectar diferentes páginas web o recursos externos, facilitando la navegación.

Estructura y elementos de HTML

    Estructura HTML básica

    <!DOCTYPE html>
                <html>
                  <head>
                    <meta charset="UTF-8">
                    <title>Mi Página</title>
                  </head>
                  <body>
                    <h1>Hola mundo</h1>
                  </body>
                </html>

Elementos esenciales

  1. Encabezados: <h1> a <h6>
  2. Párrafos: <p>
  3. Imágenes: <img>
  4. Listas: <ul>, <ol>, <li>
  5. Enlaces: <a>
  6. Tablas: <table>, <tr>, <td>
  7. Formularios: <form>, <input>
