¿Qué es <JavaScript>?

JavaScript es un lenguaje de programación utilizado para añadir características interactivas a una página web creada con HTML. Fue inventado por Brendan Eich, cofundador de Mozilla.

¿Para qué sirve JavaScript

Gracias a JavaScript puedes:

  • Interactuar con el usuario
    • Muestra u oculta elementos, valida formularios antes de enviarlos, o detecta acciones del usuario.

  • Manipular el contenido de la página
    • Cambiar texto, estilos, imágenes o cualquier otro contenido sin recargar la página.

  • Crear aplicaciones web completas
    • En el frontend (navegador), junto con HTML y CSS.

  • Crear animaciones y efectos visuales
    • Genera transiciones, sliders, galerías de imágenes, etc.

¿Cómo se usa JavaScript con HTML y CSS?

  1. Modificar contenido HTML dinámicamente:
    2. <script>
      document.getElementById("titulo").innerText = "Nuevo título";
    </script>

  2. Cambiar estilos CSS desde JavaScript:
    3. <script>
      document.querySelector("h1").style.color = "red";
    </script>

  3. Añadir o quitar clases CSS:
    4. <script>
      document.getElementById("caja").classList.add("resaltado");
    </script>

  4. Responder a eventos del usuario:
    5. <button onclick="alert('¡Haz hecho clic!')">Haz clic aquí</button>

  5. Validar formularios antes de enviarlos:
    6. <script>
      function validar() {
        let nombre = document.getElementById("nombre").value;
        if (nombre === "") alert("El nombre es obligatorio");
      }
    </script>

Cómo hacer un cuestionario de Autoevaluación

  • Caja de texto (respuesta corta):
    • <label for="respuestaCorta">1. ¿Cuál es la capital de Francia?</label>
    <input type="text" id="respuestaCorta" name="respuestaCorta" placeholder="Escribe tu respuesta">
    Validación con JavaScript:
    <script>
      if (document.getElementById("respuestaCorta").value.toLowerCase() !== "parís") {
        alert("Respuesta incorrecta");
      }
    </script>

  • Radio Button (test de única respuesta):
    • <label>2. ¿Cuál es un lenguaje de programación?</label>
    <input type="radio" name="pregunta2" value="HTML"> HTML
    <input type="radio" name="pregunta2" value="JavaScript"> JavaScript
    <input type="radio" name="pregunta2" value="CSS"> CSS

  • Combo Box (desplegable, test única):
    • <label for="combo">3. Selecciona un navegador web:</label>
    <select id="combo" name="combo">
      <option value="">--Selecciona--</option>
      <option value="Chrome">Chrome</option>
      <option value="Photoshop">Photoshop</option>
      <option value="Firefox">Firefox</option>
    </select>

  • Check Button (test múltiple):
    • <label>4. ¿Cuáles son sistemas operativos?</label>
    <input type="checkbox" name="so" value="Windows"> Windows
    <input type="checkbox" name="so" value="Linux"> Linux
    <input type="checkbox" name="so" value="Excel"> Excel

  • Selection Box (selección múltiple):
    • <label for="selMultiple">5. Selecciona lenguajes de programación:</label>
    <select id="selMultiple" name="lenguajes" multiple size="4">
      <option value="Python">Python</option>
      <option value="Word">Word</option>
      <option value="Java">Java</option>
      <option value="PowerPoint">PowerPoint</option>
    </select>

  • Decoración con CSS:
    • <style>
      input, select {
        margin-bottom: 10px;
        border-radius: 5px;
        padding: 5px;
      }
      label {
        font-weight: bold;
      }
    </style>

  • Validación con JavaScript:
    • <script>
      function validarFormulario() {
        let respuesta = document.getElementById("respuestaCorta").value;
        if (respuesta.trim().toLowerCase() !== "parís") {
          alert("Pregunta 1 incorrecta");
          return false;
        }
        alert("Formulario validado correctamente");
        return true;
      }
    </script>
