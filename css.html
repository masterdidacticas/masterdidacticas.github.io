¿Qué es <CSS>?

CSS (Cascading Style Sheets, en español, Hojas de Estilo en Cascada) es un lenguaje que se usa para definir y controlar el aspecto visual de una página web creada con HTML. Con CSS puedes cambiar colores, tamaños, tipografías, posiciones y mucho más, logrando que tu página sea visualmente atractiva y agradable para los usuarios.
¿Para qué sirve CSS?
Gracias a CSS puedes:
  • Cambiar colores y fondos
    •  Define colores para textos, fondos y elementos gráficos, haciendo tu página más atractiva.
  • Modificar tipos y tamaños de letra
    •  Elige tipografías específicas y ajusta su tamaño y estilo para mejorar la legibilidad y estética de tu página.
  • Definir el tamaño y posición de los elementos
    •  Decide cómo se organizan las imágenes, textos, botones y otros elementos dentro de la página.
  • Crear efectos visuales atractivos
    •  Añade bordes, sombras, transparencias y efectos dinámicos que hacen tu web más interactiva y moderna.
  • Adaptar tu web a cualquier dispositivo (responsive)
    •  Con CSS es fácil crear una web adaptable a ordenadores, tablets o móviles, asegurando una buena experiencia de usuario en cualquier pantalla.

    ¿Qué opciones hay para vincular CSS con tu documento HTML?


           
                    1 Crea un archivo .css externo y vincúlalo a tu HTML.                
                        <head>
                          <link rel="stylesheet" href="estilos.css" >
                        </head>                
               
       
           
                    2Usa la etiqueta <style> dentro del documento HTML.                
                        <head><style>h1 {color:blue;}</style></head>                
               
       
           
                    3Aplica estilos directamente sobre los elementos HTML.
    Se recomienda evitar esta opción en la mayoría de los casos.                    
                        <h1 style="color:blue;">Titulo azul</h1>
